Adapter

ary
创建一个最多接受n个参数的函数，多余的参数将被忽略。
使用Array.prototype.slice(0,n)和扩展运算符(...)，调用提供的函数fn，并且最多有n个参数。

const ary = (fn, n) => (...args) => fn(...args.slice(0, n));

// examples
const firstTwoMax = ary(Math.max, 2);
[[2, 6, 'a'], [6, 4, 8], [10]].map(x => firstTwoMax(...x)); // [6, 6, 10]

Tips：多个连续箭头就是es6的多次柯里化的写法

wiki 的柯里化定义：把接受多个参数的函数变换成接受一个单一参数的函数，并且返回（接受余下的参数而且返回结果的）新函数的技术。

n个连续箭头组成的函数实际上就是柯里化了 n - 1 次

前 n - 1 次调用，其实是提前将参数传递进去，并没有调用最内层函数体，最后一次调用才会调用最内层函数体，并返回最内层函数体的返回值。

call
接受一个键值和一组参数，在给定上下文时调用它们。主要用于合并。

使用闭包调用上下文中key对应的值，即带有存储参数的函数。

const call = (key, ...args) => context => context[key](...args);

// examples（神仙代码，30s看懂是不存在的，我还是太菜了）
Promise.resolve([1, 2, 3])
    .then(call('map', x => 2 * x))
    .then(console.log); // [2, 4, 6]
const map = call.bind(null, 'map');
Promise.resolve([1, 2, 3])
    .then(map(x => 2 * x))
    .then(console.log); // [2, 4, 6]

collectInto
将使用数组为参数的函数修改为使用可变参数的函数。

const collectInto = fn => (...args) => fn(args);

// example
const p1 = Promise.resolve(1);
const p2 = new Promise(resolve => setTimeout(resolve, 2000, 2));
const Pall = collectInto(Promise.all.bind(Promise));
Pall(p1, p2).then(console.log); // 2 秒后输出 [1, 2]

flip
以一个函数作为参数，然后把第一个参数作为最后一个参数。

const flip = fn => (first, ...rest) => fn(...rest, first)

over
创建一个函数，这个函数可以调用每一个被传入的并且才有参数的函数，然后返回结果。

const over = (...fns) => (...args) => fns.map(fn => fn.apply(null, args));

// examples
const minMax = over(Math.min, Math.max);
minMax(1, 2, 3, 4, 5); // [1, 5]
