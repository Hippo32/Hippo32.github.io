hexo + githubPage 搭建个人博客

试过用CSDN写过博客，用MarkDownPad2来记笔记，也尝试过有道云笔记，印象笔记等等。虽然都各有特色，各有优缺点，但就是缺少一个汇总，之前在网上看到很多别人自己搭建的博客，觉得很好看，很炫酷，所以一直都很想自己搭一个。2019的目标就是整理以前的笔记，刚好借此机会搭建一个博客并汇总一下以前的笔记吧。

初步的搭建是参考这篇文章的：使用Hexo+Github一步步搭建属于自己的博客（基础）讲得很详细，这里就不多copy一遍啦~

简单记录一下自己的搭建过程

  1. 首先，新建项目，项目名：账户名.github.io。并且勾选Initialize this respository with a README。

  2. 接着，安装hexo

    npm install hexo -g
hexo -v
hexo init
npm install
hexo g // 生成generate的意思
hexo s // 启动服务server，在服务器运行的意思

  3. 将Hexo与Github page联系起来，因为ssh这些以前也捣鼓过了，所以直接配置Deployment，在放置该项目的文件夹中找到_config.yml文件，在末尾修改deploy

    deploy:
    type: git
    respository: https://github.com/xxx/xxx.github.io.git（这个视每个人的情况不同而不同）
    branch: master

  4. 安装一个扩展：npm install hexo-deployer-git --save，这样使用编辑器编好文章后，就可以使用命令：hexo d -g，生成以及部署了。

一些常用指令

hexo new "postName"         // 新建文章
hexo new page "pageName"    // 新建页面
hexo g                        // 生成静态页面至public目录
hexo server                   // 开启预览访问端口（默认端口4000，'ctrl + c'关闭server
hexo deploy                    // 将.deploy目录部署到Github

常用组合

hexo clean
hexo g
hexo d

hexo d -g #生成部署
hexo s -g #生成预览

主题配置

选择Scheme

目前NexT支持三种Scheme，它们是：

  • Muse - 默认Scheme，这是NexT最初的版本，黑白主调，大量留白

  • Mist - Muse的紧凑版本，整洁有序的单栏外观

  • Pisces - 双栏Scheme，小家碧玉似的清新

    scheme: Mist

    选择语言

    language: zh-Hans

    设置菜单

  1. 设定菜单内容，对应的字段是menu。菜单内容的设置格式是：item name: link。其中item name是一个名称，这个名称并不直接显示在页面上，它将用于匹配图标以及翻译。

    menu:
    home: /
    archives: /archives
    #about: /about
    #categories: /categories
    tags: /tags
    #commonweal: /404.html

    设置首页不显示全文

  • 进入项目的themes/next目录
  • 打开_config.yml文件
  • 搜索auto_excerpt，把enable: false改成enable: true就可以了
  • 最后hexo d -g

两种方式来新建博文

命令形式

hexo new article （这里的article写上你的文章的名称）

输入这样的命令之后你的source/_posts下就会生成一个article.md文件，在这个文件下就可以写上你的博客内容了。用Markdown的语法去写。

直接新建方式

直接点的方式就是直接在source/_posts新建一个Markdown文件，其实和命令形式是一个道理，只不过命令形式用了命令来创建。

给你的文章打上标签

打标签能让你的文章方便检索

打开标签功能：

hexo new page tags

这时候你的source/下生成tags/index.md文件，我们将其打开，然后把它改成：

type: "tags"
comments: false

这时候你要为你的文章打上标签就可以在文章的头部写上：

tags:
    - Tag1
    - Tag2
    - Tag3

创建文章分类

hexo new page categories
