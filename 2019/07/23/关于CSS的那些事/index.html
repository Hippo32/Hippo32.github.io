关于CSS的那些事

发表于 2019-07-23 更新于 2019-07-24 分类于 CSS

经常会看到一些关于css的知识点，不记下来浑身不自在，担心我这鱼的记忆转眼就忘了，所以就有了这篇文章的存在。不是干货，纯属查漏补缺。

  • 关于CSS的不同浏览器的兼容性，应该把标准语法排在最后。

  • 提供回退机制是一种很好的做法，这样可以确保你的网站不会在低版本浏览器中挂掉。比如想弄一个渐变色的背景，应该在前面添加一行实色背景的声明。

  • 添加实色的一个好方法是取渐变色的平均色值。

    // 代码出自《CSS揭秘》
background: rgb(255, 128, 0);
background: -moz-linear-gradient(0deg, yellow, red);
background: -o-linear-gradient(0deg, yellow, red);
background: -webkit-linear-gradient(0deg, yellow, red);
background: linear-gradient(90deg, yellow, red); // IE9及之前的版本不支持渐变

  • CSS.supports

    JavaScript里提供了window.CSS.supports方法。CSS.supports的接口规范提供两种调用方法。第一种方法是使用两个参数：一个是属性名，另一个是属性值：

    var supportsFlex = CSS.supports("display", "flex"); // true

    第二种用法是简单的提供整个需要分析的样式字串：

    var supportsFlexAndAppearance = CSS.supports("(display: flex) and (-webkit-appearance: caret)"); // false

  • 如果要检测某个样式属性是否被支持，核心思路就是在任一元素的element.style对象上检查该属性是否存在。
